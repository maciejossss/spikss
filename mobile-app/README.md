# 📱 APLIKACJA MOBILNA PWA

## Pliki w tym folderze:

- `index.html` - Główna aplikacja mobilna
- `manifest.json` - Manifest PWA
- `sw.js` - Service Worker dla offline
- `install-instructions.md` - Szczegółowe instrukcje

## 🖼️ IKONY (do dodania ręcznie):

Utwórz następujące ikony w tym folderze:

### `icon-192.png` (192x192px)
- Niebieskie tło (#2563eb)
- Biała ikona narzędzi/klucza
- Format PNG

### `icon-512.png` (512x512px)
- Taka sama ikona co 192px
- Większa rozdzielczość
- Format PNG

## 🚀 Jak używać:

1. Uruchom aplikację desktop (`npm run dev`)
2. Na telefonie wejdź na: `http://192.168.1.27:5174`
3. Zainstaluj jako PWA
